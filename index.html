Requests sent

Place pixel

42["p",[X,Y,Color,1]]
P.S. You can see the same packet from server, that’s how you see pixels others place, and there might be multiple arrays at same packet when you get it from server

Send message to chat

42["chat.message",{"text":"","mention":"","type":"global","target":"","color":0}]

  • Text - self explanatory
  • Mention - the person you mentioned (you do it like "mention":"Renko" without the @)
  • Types - three possible options, global, guild, whispers - you need to
    specify a target if you are going to use whispers
  • Color - the color of your name (in format of color code)

User info

42["user.profile","name"]

This returns:
42["user.profile",{"online":true, "color":32, "painting":7, "username":"name", "x":0,"y":0, "rainbow":false, "xmas":false, "name":"Pixels World War", "slug":"pixels-world-war"}]

  • online => self explanatory
  • color => Color code of their selected color
  • painting => ID of the painting they are in (this could’ve been 17721 if it was something like this https://pixelplace.io/17721-drew-durnil-discord-map, but in this case it is 7 because pixel world war)
  • username => self explanatory
  • x and y => last pixel they placed on the painting (I don’t have premium so can’t verify this)
  • rainbow => do they have rainbow name?
  • xmas => ?
  • name => name of the painting they are in
  • slug => ID of the painting (as a name)

Join guild

POST /api/post-guild-access.php
There are 2 requests sent while joining a guild for some reason

  • id=PYTH&apply=true => first one
  • username=Tr433&guildTag=PYTH&add=true => second one

Leave guild

POST /api/post-guild-access.php
leave=true

Requests from server

Chat message

42["chat.user.message",{"username":"Tr432","color":35,"guild":"PYTH","message":"test","admin":false,"mod":false,"chatmod":false,"premium":false,"icons":[],"rainbow":false,"xmas":false,"channel":"whispers","mention":"","target":"Tr433","type":"chat","createdAt":"2022-05-21T09:57:41Z","posX":1250,"posY":1044,"posS":1}]

  • Username => self-explanatory
  • Color - Color code of the color they selected
  • Guild - Shortened name (ID?) of guild
  • Admin, mod, chatmod, premium => Icons?
  • Rainbow => Rainbow name?
  • Channel => The type you’ve seen in the message section (whispers, global, guild)
  • Icons => Array of icons (bread icon and et cetera)
  • Target => Which player whisper is sent to
  • posX => /here
  • posY => /here
  • posS => Amount of zoom (you can see it in the URL)
